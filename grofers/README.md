# Grofers Simple Scrapping
The Jupiter Notebook contains simple yet powerful and working script of Grofers Data Scrapping using Beautiful Soup library.

## Collects
Category,
Product Name,
UoM,
MRP,
Selling Price,
Club Price

### Status
Last tested 21/09/2020 with VS Code. It's working.
